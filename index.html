About

CoinMeter is a web application that allows users to track information about various cryptocurrencies.

It utilizes CSS, Bootstrap and JavaScript to create an interactive user interface that fetches data from the CoinCap.io API.

Features

View a list of the most popular cryptocurrencies.

See data such as price, market cap, volume and more for each cryptocurrency, updated every 10 seconds.

Responsive design that adapts to different screen sizes.

Technologies

HTML + CSS: Used for structure, styling and layout.

JavaScript: Used for implementing interactive functionality and fetching data from the API.

CoinCap API: Used to retrieve cryptocurrency data.
